# Overview

This project explores the method of L1 Regression (also known as LASSO regression) from a linear programming perspective. 

For more information check the wiki page on this project. Found [here](https://github.com/SteffenBorgwardt/Final-Projects_Linear-Programming_Fall-2025_CU-Denver/wiki/LASSO-Regression-as-a-Quadratic-Program).
